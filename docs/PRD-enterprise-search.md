# 企业搜索功能 PRD

> 文件路径：`pages/collect/enterprise-search.html`  
> 关联组件：`assets/components/lead-detail-modal.js`  
> 数据状态：前端内存 mock 数据，未连接后端。

---

## 1. 功能说明

企业搜索是「数据采集」模块下的高级企业筛选页面。用户可通过可视化的复合条件构建器，从工商信息、联系方式、业务信息、产品信息 4 类字段中自由组合筛选条件，支持条件组嵌套，实时查看符合条件的企业卡片列表，并点击卡片查看企业详情。

核心能力：

- 复合高级筛选：支持「满足下列 所有/任一」+ 多条件行 + 嵌套条件组。
- 字段池选择：按 4 大分类展示字段，支持关键词搜索。
- 多种字段类型：文本、日期、数值、下拉多选、联系人/联系电话。
- 结果展示：卡片式列表，展示企业关键信息 + 联系方式数量标签。
- 详情查看：点击卡片调用公共 `LeadDetailModal` 查看完整企业信息与关联联系人。
- 排序：支持按成立日期、注册资本、客户等级排序。

---

## 2. 操作入口

- 顶部菜单/侧边栏：**数据采集 · 企业搜索**
- 直接访问地址：`pages/collect/enterprise-search.html`
- 权限校验：由 `assets/app.js` 的 `localStorage` 登录守卫控制，未登录跳转 `login.html`。

## 3. 交互流程

### 3.1 进入页面

1. 用户点击侧边栏「企业搜索」菜单。
2. 页面加载默认筛选条件：根条件组（满足所有）+ 1 个空的「请选择类型」条件行。
3. 结果区默认展示全部企业卡片，按综合排序排列。

### 3.2 添加/编辑筛选条件

1. 点击条件行的「请选择类型」按钮，弹出「选择筛选字段」弹窗。
2. 弹窗左侧为 4 大分类 tab：工商信息、联系方式、业务信息、产品信息。
3. 右侧展示当前分类下的字段；顶部搜索框支持按关键词实时过滤所有分类字段。
4. 点击字段后弹窗关闭，条件行显示：字段名 + 匹配方式下拉 + 值输入组件。
5. 用户切换匹配方式或输入值后，条件即时生效（值变更不自动触发搜索，点击「搜索」后执行）。

### 3.3 添加条件组

1. 点击「添加单个条件」：在当前组末尾新增 1 个空条件行。
2. 点击「添加条件组合」：在当前组末尾新增一个子条件组，子组自带「满足下列 所有/任一」下拉，并默认包含 1 个空条件行。
3. 非根条件组右上角显示「删除组合」按钮，点击后移除该组及其所有子条件。

### 3.4 执行搜索与排序

1. 条件构建完成后点击「搜索」，结果区展示匹配企业数量与卡片列表。
2. 结果为空时显示「未找到符合条件的企业」。
3. 通过右上角排序下拉切换：综合排序 / 按成立日期 / 按注册资本 / 按客户等级。
4. 点击「清空」：重置为初始的根条件组 + 1 个空条件行，并重新搜索。

### 3.5 查看详情

1. 点击企业卡片或「查看详情」按钮，打开公共线索详情弹窗。
2. 弹窗左侧展示企业工商/业务字段，右侧展示关联联系人卡片。
3. 联系人卡片显示：姓名、岗位（右上角标签）、电话、邮箱、地址、数据来源、来源网页、采集器；点击卡片或来源网页可跳转固定来源页。

## 4. 核心规则

### 4.1 字段分类与类型

| 分类 | 字段 | 类型 | 值输入组件 |
|---|---|---|---|
| 工商信息 | 客户名称、统一社会信用代码、法定代表人、注册地址、省市区、曾用名、登记机关、纳税人识别号、工商注册号、组织机构代码 | text | Tag 输入 |
| 工商信息 | 成立日期、营业期限 | date | 日期选择框 |
| 工商信息 | 注册资本、实缴资本、企业规模、参保人数、年营业额 | number | 数值输入框 |
| 工商信息 | 企业类型、企业状态 | select | 标签式多选下拉 |
| 联系方式 | 联系人、联系电话 | contactName / contactPhone | Tag 输入 |
| 业务信息 | 客户等级、优先级、行业类型、采集器、数据来源 | select | 标签式多选下拉 |
| 业务信息 | 项目名称、项目编号 | text | Tag 输入 |
| 业务信息 | 采集时间 | date | 日期选择框 |
| 产品信息 | 经营范围、关键需求摘要 | text | Tag 输入 |

### 4.2 匹配方式规则

| 字段类型 | 可选匹配方式 | 说明 |
|---|---|---|
| text / contactName / contactPhone | 包含任一、包含全部、不等于 | 多个关键词以回车生成 Tag，按 OR / ALL / NOT 匹配 |
| date | 等于、大于、小于、在...之间 | 「在...之间」显示开始时间、结束时间两个日期框 |
| number | 等于、大于、小于、大于等于、小于等于 | 按数值比较 |
| select | 等于、不等于 | 多选时按「等于任一 / 不等于任一」匹配 |

### 4.3 Tag 输入规则（文本/联系人/联系电话）

- 单个词不超过 30 个字符。
- 最多 20 个词。
- 回车键生成一个 Tag。
- 点击 Tag 上的 × 可删除。
- 点击复制按钮可复制所有 Tag（以换行分隔）。

### 4.4 日期区间规则

- 选择「在...之间」时，自动显示两个日期输入框，中间以「至」连接。
- 选择框上方不显示「开始时间/结束时间」文案。
- 开始日期为空或结束日期为空时，该侧条件视为不限。

### 4.5 下拉多选规则

- 枚举类字段使用标签式多选组件：
  - 选中项以 Chip 形式展示在选择框内，可点击 × 移除。
  - 点击下拉箭头展开选项列表，已选项显示对勾。
  - 选择框宽度随标签内容自适应，标签过长时自动换行。
- 匹配逻辑：
  - 等于：企业字段值等于所选任意一个选项即命中（OR）。
  - 不等于：企业字段值不等于所有选中项才命中。

### 4.6 条件组逻辑

- 每个条件组顶部有「满足下列 所有/任一」下拉：
  - 所有：组内所有条件/子组必须同时满足（AND）。
  - 任一：组内任一条件/子组满足即可（OR）。
- 支持无限层级嵌套，子组左侧显示缩进与连接线。
- 根条件组不可删除；非根条件组可删除。

### 4.7 条件计数

- 页面顶部显示「已选择：N 个筛选条件」。
- 仅统计已选择具体字段的条件行；空「请选择类型」行不计入。

### 4.8 结果排序

| 排序项 | 规则 |
|---|---|
| 综合排序 | 保持原始数据顺序 |
| 按成立日期 | 成立日期由新到旧 |
| 按注册资本 | 按注册资本数值由高到低 |
| 按客户等级 | A > B > C > D |

## 5. 状态流转

### 5.1 筛选面板状态

```
初始状态
  │
  ▼
根条件组（满足所有）+ 1 个空条件行
  │
  ├── 点击「请选择类型」→ 打开字段池弹窗 → 选择字段 → 条件行进入「已配置」状态
  │
  ├── 点击「添加单个条件」→ 当前组追加 1 个空条件行
  │
  ├── 点击「添加条件组合」→ 当前组追加 1 个子条件组（含 1 个空条件行）
  │
  ├── 切换「所有/任一」→ 当前组逻辑变更，不影响条件数量
  │
  ├── 点击删除按钮 → 移除对应条件行/子条件组
  │
  ▼
点击「搜索」→ 条件进入「生效」状态，结果区刷新
  │
  ▼
点击「清空」→ 重置为初始状态
```

### 5.2 条件行状态

| 状态 | 表现 | 触发方式 |
|---|---|---|
| 未配置 | 显示「请选择类型」虚线按钮 | 新增条件行 / 清空 |
| 已配置 | 显示字段名 + 匹配方式 + 值输入组件 | 选择字段后 |
| 已生效 | 参与搜索结果过滤 | 点击「搜索」 |

### 5.3 字段池弹窗状态

| 状态 | 表现 |
|---|---|
| 分类浏览 | 左侧分类高亮，右侧展示该分类字段 |
| 关键词搜索 | 顶部输入关键词，右侧展示跨分类匹配字段，并显示所属分类 |
| 选择字段 | 点击字段后关闭弹窗，目标条件行被赋值 |

### 5.4 结果区状态

| 状态 | 表现 |
|---|---|
| 有结果 | 展示企业卡片列表，顶部显示命中数量 |
| 无结果 | 展示「未找到符合条件的企业」空态 |
| 排序切换 | 列表按所选规则重新排列，不重置筛选条件 |

### 5.5 详情弹窗状态

| 状态 | 表现 |
|---|---|
| 关闭 | 页面仅展示列表 |
| 打开 | 弹窗展示企业基本信息 + 关联联系人 |
| 联系人交互 | 点击联系人卡片/来源网页跳转固定来源页；点击来源按钮打开数据来源网页弹窗 |

---

## 6. 异常与边界

- 未选择字段的条件行不参与搜索。
- 条件值为空时，该条件视为不限。
- 数值字段输入非数字时，按空值处理。
- 清空操作不会保留历史条件，直接重置为初始状态。
- 页面刷新后筛选条件丢失（仅内存状态，未持久化）。
